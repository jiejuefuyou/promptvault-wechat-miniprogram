## 1️⃣ 名称（4-30 字符必填）

**首选**:
```
AI 提示词小抄 PromptVault
```
✓ 13 字符
✓ 含「AI 提示词」精准搜索匹配
✓ 「小抄」自带轻松感，不像工具书

**备选**（首选被占用就上）:
```
A. PV 提示词小抄
B. 提示词速记 PV
C. AI 提示词随身带
```

**避免**:
- ❌ 仅英文 (PromptVault)：中文用户搜不到
- ❌ "ChatGPT 提示词"：踩 OpenAI 商标
- ❌ 含具体数字（如 "113 条"）：内容增加每次都要改名/简介，麻烦

**规格**:
- **144 × 144 像素**最小，**512 × 512** 推荐
- JPG / PNG，< 2MB
- 不能含二维码 / 特殊字符

**简单方案 3 选 1**:

**方案 A — 在线生成（最快，5 分钟）**:

prompt 直接抄:
```
Minimalist app icon, letter "P" in center, purple gradient #6c4be0 to #c084fc,
dark navy background, clean modern design, square 1:1, no text, app icon style
```

**我推荐**：先用方案 C 上线，赚到第一个真实用户再优化头像。**别为完美的图标推迟上线**。
